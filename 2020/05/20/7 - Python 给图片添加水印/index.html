【python】给图片添加水印

发布于 2020-05-20 in python
本文字数：516 字 | 阅读时长 ≈ 2 分钟

水印

给图片添加水印的方式有两种：一、添加文字水印 二、添加图片水印

添加文字水印

from PIL import Image, ImageDraw, ImageFont

def add_num(img):
    draw = ImageDraw.Draw(img)
    myfont = ImageFont.truetype('C:/windows/fonts/Arial.ttf', size=40)
    fillcolor = "#ff0000"
    width, height = img.size
    draw.text((width-50, 5), '99', font=myfont, fill=fillcolor)
    img.save('result.jpg','jpeg')

    return 0

if __name__ == '__main__':
    image = Image.open('F:/Picture/touxiang.jpg')
    add_num(image)
    image.show()

添加图片水印初级

from PIL import Image, ImageDraw, ImageFont

def add_num(im,mark):
    layer = Image.new('RGBA', im.size, (0, 0, 0, 0))
    layer.paste(mark, (im.size[0] - 150, im.size[1] - 150))
    out = Image.composite(layer, im, layer)
    out.save('result.jpg','jpeg')
    out.show()
    return 0

if __name__ == '__main__':
    image = Image.open('F:/Picture/touxiang.jpg')
    mark = Image.open('F:/Picture/1022.png')
    add_num(image,mark)

添加图片水印高级（支持水印格式为 svg、png ···）

上述通常添加水印的方式有以下缺点：

  1. 水印图片格式不支持 svg
  2. 无法设置水印的透明度
  3. 直接将水印粘贴到原图上后，新生成图片体积变大很多

  • 安装包 wand

    $ apt-get install libmagickwand-dev
$ pip install Wand

  • 添加水印

import subprocess
from wand.image import Image as WandImage
# 添加水印函数

  :param origin_img_path: 原图路径
  :watermark_path: 水印途径
  :new_img_path: 添加水印后的图片 
  :param blend_mode: 图片合成方式
  :param ratio: 水印占整个图片的比例
  :param transparency: 水印透明度
  :return: relative_path: 图片相对路径
  """
  source_obj = WandImage(filename=origin_img_path)
  watermark_obj = WandImage(filename=watermark_path)

  if ratio > 1:
      ratio = 1
  # 将水印图缩小
  watermark_width = int(source_obj.size[0] * ratio)
  watermark_height = int(float(watermark_width) / float(watermark_obj.size[0]) * watermark_obj.size[1])
  # 如果水印格式为 svg，则需先转换
